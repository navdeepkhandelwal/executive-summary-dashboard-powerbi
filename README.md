# Executive Summary Dashboard (Power BI)

## Project Overview
This project presents a high-level business performance dashboard built using Power BI.  
The dashboard provides an executive-level overview of key business metrics including total sales, profit, order trends, and category performance.

It enables management to quickly monitor business performance and make data-driven decisions.

---

## Key Features
- High-level KPI tracking
- Revenue and profit monitoring
- Category performance analysis
- Sales trend visualization
- Interactive filters and slicers for dynamic insights

---

## Tools Used
Power BI  
DAX  
Excel  
Data Visualization  

---

## Key Insights
- Identified top performing product categories
- Highlighted revenue and profit trends over time
- Provided executive-level view of overall business performance
